# Quotation Service

Приложение будет доступно на порту **8080**.

## Быстрый запуск

Для полной инициализации проекта выполните:
```bash
make init
```

## Пошаговый запуск

1. Запуск образов:
   ```bash
   make up
   ```

2. Миграции:
   ```bash
   make migrate-up
   ```

3. Запуск приложения:
   ```bash
   make run
   ```

## Swagger (Документация)

После запуска документация доступна по адресу: **[http://localhost:8080/docs](http://localhost:8080/docs)**
